## Description
A menu-driven Python calculator application that performs basic and advanced arithmetic operations.  
The calculator also maintains a history of calculations, allows clearing the history, and handles errors gracefully.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division (with division-by-zero handling)
- Power operation
- Modulus operation (with zero-check)
- View calculation history
- Clear calculation history
- Continuous execution until user exits

---

## Concepts Used
- Python functions
- Conditional statements
- Loops
- Lists (for storing history)
- Exception handling (`try-except`)
- String formatting (f-strings)

---

## How It Works
1. User selects an operation from the menu
2. Inputs two numbers
3. Calculator performs the operation
4. Result is displayed
5. Successful calculations are stored in history
6. User can view or clear history at any time
